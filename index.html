    RePho

    RePho is a flexible re-photography tool written in Python and QML. And it does time-lapse photography too ! :)

    How to use

    • open an old image
    • take pictures using the old image as a guide

    Feature overview

    • open old images from various sources
      • from the on-device gallery
      • image files
      • Urls
      • capture directly with the camera
    • time-lapse photography made easy
      • click on the clock icon, select an interval and you are ready to go :)
      • works even when screen is turned off
    • the old image can be rotated and its opacity varied
    • images are named in a nice fixed-size ascending order
    • compare the images you have just taken
      • accessible using the compare button in the quick menu
